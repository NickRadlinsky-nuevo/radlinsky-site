# Прототип посадочной страницы для акций НЦ «Аполлонія»

**Эталон:** [`/summer-sale`](https://radlinsky.com.ua/summer-sale) (файл `summer-sale.html`)
**Шаблон с токенами:** `_templates/promo-landing.html`

Одностраничный промо-лендинг под акции (скидки на онлайн-продукты, наборы МК, интенсивы).
Самодостаточный HTML-файл: все стили инлайн, зависимости — только Google Fonts и картинки из `/images/`.

## 1. Структура блоков (сверху вниз)

| # | Блок | Обязателен | Что внутри |

| 1 | `header.site` | ✅ | Шапка как на основном сайте: лого, меню, соцсети, бургер. **Не менять** |
| 2 | `header.hero` | ✅ | Ярлык акции, H1, подзаголовок, имена лекторов, 3 чипа, цена (−X%, новая, старая зачёркнутая), CTA-кнопка, статичная картинка лекторов справа |

## 2. Токены шаблона

| Токен | Пример (summer-sale) |
|---|---|
| `{{SLUG}}` | `summer-sale` — URL страницы и имя og-картинки |
| `{{PAGE_TITLE}}` | `6 онлайн майстер-класів зі знижкою −60%` |
| `{{PAGE_DESCRIPTION}}` | 1–2 предложения, ~150 символов |
| `{{OG_TITLE}}` / `{{OG_DESCRIPTION}}` | для превью в мессенджерах |
| `{{PROMO_LABEL}}` | `літня пропозиція` (ярлык в hero и в карточке цены) |
| `{{H1}}` | `6 онлайн майстер&#8209;класів` (`&#8209;` = неразрывный дефис) |
| `{{HERO_SUBTITLE}}` | 1 предложение, до ~120 символов |
| `{{LECTOR_1..3}}` | имена; если лекторов ≠ 3 — правьте строку `.names` |
| `{{CHIP_1..3}}` | `30 годин відео` / `15 балів БПР` / `доступ 3 місяці` |
| `{{DISCOUNT}}` | `60` (без знаков) |
| `{{PRICE}}` / `{{OLD_PRICE}}` | `2480` / `6200` |
| `{{PAYMENT_URL}}` | ссылка WayForPay (встречается 2 раза — hero и offer) |
| `{{HERO_IMAGE}}` / `{{HERO_IMAGE_ALT}}` | имя PNG в `/images/` + alt |

## 5. Правила контента

- **Программы МК** — только реальные тексты от владельца, ничего не выдумывать (медицина!)
- Длинные программы прятать в `<details><summary>Повна програма</summary>` с подзаголовками `<h4>` (Теоретичний блок / Практичний блок / Демонстрація…)
- Локации и тайминги очных событий в программах онлайн-записей — убирать
- **FAQ-ответы (проверенные владельцем, июль 2026):**
  - Доступ: «менеджер зв'яжеться протягом робочого дня» (НЕ «одразу після оплати»)
  - БПР: тестирование после обучения, сертификат — в течение месяца после тестирования
  - Возврат средств: пункт про возврат НЕ включать
- Под CTA-кнопкой в hero подписей не ставить
- Цена: всегда плашка −X% + новая цена крупно + старая зачёркнутая

## 6. Чеклист деплоя

1. Скопировать `_templates/promo-landing.html` → `<slug>.html` в корень репо
2. Заменить все `{{ТОКЕНЫ}}` (проверить: `grep -o '{{[A-Z_0-9]*}}' <slug>.html` — должно быть пусто)
3. Положить картинки: `images/<hero>.png`, `images/og-<slug>.jpg` (1200×628, можно экспорт баннера из Figma)
4. `noindex` оставить, в sitemap.xml НЕ добавлять, с сайта НЕ линковать (страница только для рекламы/рассылок) — если владелец не решил иначе
5. Проверить локально: `open <slug>.html`
6. `git add <slug>.html images/... && git commit && git push` → Cloudflare Pages задеплоит на `radlinsky.com.ua/<slug>` за ~1 мин
7. Проверить live: страница 200, картинки 200, обе кнопки ведут на WayForPay, превью ссылки в Telegram
